# Leafhopper_flowering_fields

Data and code for:
# Structurally complex and mature farmland flower fields maximise leafhopper abundance and diversity
## Content of the repository

    Data: the folder data contains
        Raw data of the sites variables, leafhopper taxon abundances (.csv/.xlsx)
        Raw data of the ecological guilds (ecotype, LifeStrategy, etc.) of each leafhopper taxon (.csv/.xlsx)
        Raw data of taxon abundances for NMDS (.csv/.xlsx)
